## 4. Functional Requirements

* Sticky WhatsApp button (visible on all pages)
* Mobile-first responsive layout
* Fast loading speed
* Clean navigation
* Clear CTA on every section

---

## 5. UI Behavior Rules

* No unnecessary animations
* Smooth transitions only
* Maintain strong visual hierarchy
* Avoid clutter

---

## 6. Development Notes

* Build sections as reusable components
* Keep layout modular
* Avoid hardcoding content
* Optimize for performance

---

## 7. Critical Guidelines

* Do not add generic phrases (e.g., “best solutions”, “innovative tech”)
* Do not overload sections with text
* Keep messaging sharp and direct
* Always guide user toward contact action

---

## 8. Primary Conversion Goal

User should:
→ Click WhatsApp
→ Submit contact form
→ Call directly

Everything on this page must support that.
